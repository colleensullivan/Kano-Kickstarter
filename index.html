Welcome to Kano.

Kano is a computer you build and code yourself. Lego simple, Raspberry Pi powerful, and hugely fun.

Kano is for anyone who wants to start creating with technology – not just consuming it. It's a simple, fun, useful toolkit that you can use to start taking control of the world around you.

    What Can I Do With Kano?

  • A computer, powered by Raspberry Pi
  • Games like Minecraft, Pong, and Snake
  • Music and sounds
  • HD video
  • A speaker
  • A wireless server
  • A custom case with stickers
  • Kano is open source

    Awesome for kids with special abilities like dyslexia.

    Even if reading is challenging for you, you can build the Kano computer and code! Kano introduces kids to some Python basics, and to visual development environments by use of Kano Blocks, which allow kids to drag and drop pieces of code and invent their own version of Pong, or make some Minecraft creations. With Kano Blocks, kids can learn about programming basics, and even make their own first programs.
